# itch.io Theme Colors

Color reference for styling the itch.io page so it matches in-game visuals.

## Core Palette

| Name | Hex | Usage |
| --- | --- | --- |
| Void Black | `#000000` | Global page background, negative space |
| Storm Blue | `#296BFA` | Background energy tones |
| Nebula Cyan | `#7AD1FF` | Secondary text accents, subtle highlights |
| Scanner Cyan | `#59F2FF` | Links, interactive highlight, key UI accents |
| Relay Mint | `#33FFCC` | Primary button emphasis |
| Relay Glow | `#80FFF2` | Soft glow accents or hover states |
| CRT Magenta | `#F233D9` | Header accents and stylized highlights |
| Storm Magenta | `#F247F2` | Secondary accent for effects |
| Hazard Violet | `#CC33F2` | Optional decorative accents |
| Wreck Amber | `#FFB340` | Callouts and key objective highlights |
| Lightning Gold | `#FFF266` | Warning emphasis, caution tags |
| HUD Blue | `#408CB3` | Secondary panel lines and separators |

## Recommended itch.io Theme Mapping

Use these values in the itch.io theme editor as a starting point.

| itch.io Field | Value | Notes |
| --- | --- | --- |
| `BG` | `#000000` | Full page background |
| `BG2` | `#08111F` | Content column background |
| `BG2 Alpha` | `0.88` | Keeps readability while preserving atmosphere |
| `Text` | `#EAF7FF` | Main body text |
| `Link` | `#59F2FF` | Links and interactive text |
| `Buttons` | `#33FFCC` | Primary action buttons |
| `Headers` | `#F233D9` | Section headers and title accents |

## Optional Accent Rules

- Use `#FFB340` for small objective callouts only.
- Use `#FFF266` for warning or hazard labels sparingly.
- Keep long paragraphs in `#EAF7FF` for contrast.

## Contrast Notes

- Avoid magenta body text on dark backgrounds for long reads.
- Keep body text at high contrast against `BG2`.
- Reserve bright accents for headings, links, and key buttons.
