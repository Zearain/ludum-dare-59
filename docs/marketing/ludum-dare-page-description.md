# Ludum Dare Game Page Description

Use this copy on the Ludum Dare submission page. It is written for fast scanning by jam voters.

## Short Description

Restore drifting relays inside a violent ion storm to follow a pirate dreadnought's final transmission, then race to recover its black box with the least damage and fastest time.

## Full Description

`Ghost Frequency: Storm Relay` is a top-down neon space-tracing game built for Ludum Dare around the theme `Signal`.

Pilot a fragile hunter ship through storm sectors, pulse your scanner to read distorted feedback, and thread through hazards to stabilize relay chains. Each activated relay reveals another fragment of the vanished dreadnought `Night Saint` transmission. Survive long enough to reach the final wreck and extract the black box.

## Theme Fit (`Signal`)

- Scanning is the core mechanic, not a side feature.
- Distortion and interference make every signal reading uncertain.
- Progress comes from restoring transmission relays across sectors.

## Core Features

- Top-down arcade movement focused on route choice and survival.
- Scan pulse with cooldown to get directional objective feedback.
- Multi-stage objective chain: relay sectors into final wreck extraction.
- Hazard pressure from storm clouds, debris fields, and lightning arcs.
- Score based on completion, speed, hull integrity, and sectors cleared.

## Controls

- `WASD` / left stick: move
- `Space` / `X`: scan pulse
- Hold `E` / `A`: activate objective
- Arrows / D-pad: navigate menus
- `Enter` / `A`: confirm

## Run Length and Goal

- Expected successful run: about 12-15 minutes.
- Goal: recover the black box before your hull fails.

## Built With

- Godot 4.6 + C# (.NET)
